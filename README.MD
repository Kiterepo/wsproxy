# 📖 Introduction
A lightweight proxy tool based on the WebSocket protocol
# 🚀 Features
- Lightweight
- No configuration
- Wrap CDN
# 🔨️ Build
```shell
git clone github.com/golangboy/wsproxy
```

## Client
```shell
cd client
go build .
./client -h
```

### Server
```shell
cd server
go build .
./server -h
```
